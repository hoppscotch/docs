**Hoppscotch is an open-source API development ecosystem.** Available offline, on-prem, and on the cloud with Web, Desktop, and CLI apps — a lightweight, open-source alternative to Postman and Insomnia.

Built from the ground up with ease of use and accessibility in mind, providing all the functionality needed for API developers with a minimalist, unobtrusive UI.
